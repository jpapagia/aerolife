Getting Started with AeroLife

Deploy AeroLife quickly with this user-friendly documentation.

View on GitHub Deploy to Vercel

Installation

Run the following command to install:

npm install

Configuration

Modify the settings in the configuration file:

{
  "apiKey": "your-api-key",
  "environment": "production"
}

Running the Application

Start the application with the following command:

npm run start
