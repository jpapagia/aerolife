Understanding Air Quality and Pollutant Data

The core.html page provides real-time air quality information based on various pollutants. Each pollutant is monitored and categorized based on its concentration in the air. This data helps you understand the current air quality and its potential impact on health.

Pollutants Monitored:

  • Sulfur Dioxide (SO₂): Produced from burning fossil fuels, SO₂ can cause respiratory problems and aggravate lung diseases like asthma.
  • Nitrogen Dioxide (NO₂): Emitted from vehicle exhaust and industrial processes, NO₂ contributes to respiratory issues and forms smog.
  • Particulate Matter (PM10 and PM2.5): Tiny particles suspended in the air. PM10 includes larger particles like dust, while PM2.5 consists of fine particles that penetrate deep into the lungs.
  • Ozone (O₃): A secondary pollutant formed by chemical reactions in sunlight. High levels of ozone can irritate the respiratory system.
  • Carbon Monoxide (CO): A colorless, odorless gas produced by incomplete combustion. High levels of CO reduce oxygen delivery to the body's organs.

Air Quality Index (AQI) Thresholds:

Fair 2 [20; 80) [40; 70) [20; 50) [10; 25) [60; 100) [4400; 9400)
Moderate 3 [80; 250) [70; 150) [50; 100) [25; 50) [100; 140) [9400; 12400)

Getting Started

Welcome to Aerolife! Follow this guide to set up quickly.

Installation

Run the following command to install:

git clone https://github.com/jpapagia/aerolife && cd aerolife
npm install

Configuration

Modify the settings in the configuration file:

{
  "apiKey": "your-api-key",
  "environment": "production"
}

Running the Application

Start the application with the following command:

npm run start
